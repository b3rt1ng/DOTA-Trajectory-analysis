Pour utiliser matplotlib apparamment il faut installer d'abord quelques trucs.
Faire:
sudo apt-get install python3.8-dev
sudo apt-get install python3.8-numpy
sudo apt-get install python3-matplotlib

Pour compiler faire: make all depuis le repertoire cplusplus
Pour executer faire: bin/main

Avec vscode:
Installer l'extension c/c++
Normalment vu que j'ai upload les fichiers de configurations si ça marche chez moi ça devrait aussi le faire pour tout le monde
compte tenu du fait que l'on ait tous python3.8
Il suffira juste de cliquer sur le bouton run de vscode ou la touche F5

Notes:
  Le matplotlibcpp.h rend la compilation vraiment longue, voir quels sont les moyens de plot sans.
